## Pipeline

```
Idea → MovieEngine → MediaGen → FFmpeg
```

**Idea** — User-provided concept, theme, or story seed.

**MovieEngine** — Generates the full narrative structure: screenplay, character bibles, location design, plot arcs, and shot-by-shot storyboards.

**MediaGen** — Produces all visual and audio assets: character portraits, location references, scene frames, and video clips for every shot.

**FFmpeg** — Post-production assembly: scene concatenation, chapter overlays, title cards, credits, and platform-specific encoding.
